# Copilot Instructions

## Terminal Execution Preferences

When the user asks you to run commands or perform actions that require terminal execution:

1. Execute terminal commands directly using `run_in_terminal` without asking for permission first
2. Always provide a brief explanation of what each command does and its purpose
3. Only ask for confirmation if the command is destructive (deletes files, overwrites data, or makes permanent changes)

This allows faster iteration and reduces unnecessary back-and-forth in conversations.
